# Functions White Paper

This document mirrors the in-app Functions White Paper in Settings. It is provided for versioning and offline reference.

## Purpose and Audience
For admin staff, event coordinators, and managers. It explains the end-to-end workflow for functions.

## Workflow Overview
1) Create or open a function record.
2) Attach contacts and capture notes.
3) Add menus and build the quote.
4) Build and send a proposal to the client.
5) Track payments and remaining balance.
6) Finalize tasks and close out the function.

## Function Details (Sidebar)
1) Open a function from the Functions list.
2) Click the event name in the sidebar to edit it.
3) Set Status from the dropdown.
4) Set Event Date, Start Time, End Time.
5) Enter Group Size (drives per-person quantities).
6) Select Room / Space and Event Type.
7) Changes auto-save when you leave a field.

## Contacts
1) Click **+ Add** in the Contacts block.
2) Search or create a contact.
3) Use the contact menu to Edit, Make Primary, or Remove.
4) Primary contact is used for proposals.

Screenshot placeholder: Contacts section with add/edit actions.

## Notes
1) Open the **Notes** tab.
2) Click **New Note**.
3) Select note type (Proposal/Internal/Call).
4) Write the note and **Save**.
5) Proposal notes can be included in the Proposal Builder.

Screenshot placeholder: Notes tab with proposal note type selected.

## Tasks
1) Open the **Tasks** tab.
2) Click **Add Task**.
3) Set due date/owner if needed.
4) Tick to complete.

Screenshot placeholder: Tasks tab with a checklist in progress.

## Menus (Settings)
1) Go to **Settings → Menus**.
2) Create **Menu Categories** (Catering, Room Hire, etc.).
3) Create **Menus** (top-level menu with base price).
4) Create **Menu Choices** (default items).
5) Create **Menu Options** (priced items).
6) Create **Sales Units** (per-person or fixed).

Screenshot placeholder: Menu settings with categories, menus, and choices.

## Quote Builder (Menus)
1) Open the **Quote** tab.
2) Click **Add Menu**.
3) Choose Category and Menu, then click **Add**.
4) If missing, click **Create New Menu**, build it, then add it.
5) Adjust Qty and Price Each as needed.
6) Total = Qty × Price Each. Subtotal is the sum of totals.

Screenshot placeholder: Quote menu table with qty/price/total columns.

## Quote Menu Options (Dropdown)
1) Click **Menu options** on a menu.
2) **Edit menu** opens the menu editor.
3) **Resync prices** resets prices/qty to defaults.
4) **Client can choose** enables client selection.
5) **Hide from proposal** removes from client view.
6) **Remove menu** deletes from quote.

Screenshot placeholder: Menu options dropdown expanded.

## Refresh vs Save (Quote)
1) **Actions → Save** keeps your current edits.
2) **Actions → Refresh from menus** resets prices/qty to menu defaults.

Screenshot placeholder: Quote actions dropdown showing Save and Refresh.

## Proposal Builder
1) Scroll to **Proposal Builder**.
2) Select menus and notes to include.
3) Choose terms and templates.
4) Preview.
5) Email the client link.
6) Use **Apply client selections** after client changes.

Screenshot placeholder: Proposal builder with selections ticked.

## Payments and Discounts
1) In Quote totals, click **Actions → Edit totals**.
2) Click **Add discount**, enter amount, **Save**.
3) Add payment: description, method, date, amount.
4) Discount appears as a negative line.
5) Totals show ex GST, GST, incl GST, payments, remaining.

Screenshot placeholder: Quote totals with payments and remaining due.

## Communications
1) Open the **Communications** tab.
2) Open a message to read/reply.
3) Keep client contact in the same thread.

Screenshot placeholder: Communications view with message thread.

## Reports
1) Go to **Reports** in main navigation.
2) Choose date range and status.
3) Review revenue, cost, profit.

## Setup Checklist
1) Settings → Rooms / Spaces.
2) Settings → Event Types.
3) Settings → Menus (categories, menus, choices, units).
4) Settings → Proposal Terms.
5) Settings → Note Templates.
6) Settings → Calendar.
